This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Install library:
```bash
npm install
# or
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Run build

```bash
npm run build
npm run start
# or
yarn build
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx` - Landing page Home.

Folder `pages/blogs` - You can editing pages blogs and blog detail

Folder `pages/comics` - You can editing pages comics and comic detail

## Deploy on Server

Pull code from git

Run:

```bash
docker-compose down
docker-compose build
dokcer-compose up
```
